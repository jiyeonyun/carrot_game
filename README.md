# carrot game🥕
**벌레를 피해 당근을 클릭하는 게임입니다.**
1. 재생버튼 누를시 타이머 스타트/ 당근과 벌레는 랜덤 위치에 포지션
2. 당근 클릭시 스코어의 숫자 줄어듬
3. 벌레 클릭시 게임오버
4. 당근을 전부 클릭시 you win모달 뜸
5. replay 누를시 처음으로 리셋
